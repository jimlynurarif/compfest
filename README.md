## How To use 🧑‍💻
1. Clone this repository
```
git clone https://github.com/jimlynurarif/compfest.git
```
2. Go into the root directory
```
cd compfest
```
4. Build Docker image
```
docker-compose build
```
5. Run Docker image
```
docker-compose up
```

6. Open Website
```
localhost:5000
```
